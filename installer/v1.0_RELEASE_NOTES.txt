## üíæ Installation

**System Requirements:**
- Windows 10/11 (64-bit)
- 4GB RAM minimum (8GB recommended)
- 500MB disk space
- Internet connection (for Firebase authentication)

**What's Included:**
- Self-contained .NET 8 runtime (no separate install needed)
- Automatic VC++ Redistributable download/install
- Firebase credential configuration
- Diagnostic tools and troubleshooting guides
- Complete documentation

**Installation is Simple:**
1. Run `Desktop Food CostSetup-1.0.0-x64-Complete.exe`
2. Follow the installer wizard
3. Launch Desktop Food Cost from Start Menu or Desktop
4. Create your first user account (automatically becomes Admin)
